# 용어 사전

배차통 시스템에서 사용하는 주요 용어를 정리합니다.

---

## 업무 용어

### 기본 용어

| 용어 | 설명 |
|------|------|
| **배차** | 화물 운송 건을 기사에게 할당하는 행위. 배차통의 핵심 업무 |
| **화주** | 화물 운송을 의뢰하는 업체. 배차통의 주요 고객 |
| **차주** | 화물을 운송하는 기사. 실제 운송을 담당 |
| **운임** | 화물 운송에 대한 비용. 화주 운임과 기사 운임이 다를 수 있음 |
| **수수료** | 화주 운임에서 기사 운임을 뺀 금액. 배차통의 수익 |

### 위치 관련

| 용어 | 설명 |
|------|------|
| **상차지** | 화물을 싣는 장소. 출발지 |
| **하차지** | 화물을 내리는 장소. 도착지 |
| **상차** | 화물을 차량에 싣는 행위 |
| **하차** | 화물을 차량에서 내리는 행위 |

### 일정 관련

| 용어 | 약어 | 설명 |
|------|------|------|
| **당상** | 당일 상차 | 오늘 상차 |
| **내상** | 내일 상차 | 내일 상차 |
| **월상** | 월요일 상차 | 월요일 상차 |
| **당착** | 당일 하차 | 오늘 하차 |
| **내착** | 내일 하차 | 내일 하차 |

### 운송 유형

| 용어 | 설명 |
|------|------|
| **편도** | 출발지에서 도착지까지 한 방향 운송 |
| **왕복** | 출발지 → 도착지 → 출발지로 돌아오는 운송 |
| **혼적** | 여러 화물을 한 차량에 함께 싣는 운송 |
| **선착불** | 화물이 도착지에 도착한 후 운임을 지급하는 방식 |

### 정산 관련

| 용어 | 설명 |
|------|------|
| **청구** | 화주에게 운송 비용을 청구하는 것 |
| **청구월** | 해당 배차 건이 청구되는 월 (예: 2024-01) |
| **거래명세서** | 월별 거래 내역을 정리한 문서 |
| **세금계산서** | 부가가치세 신고를 위한 법적 증빙 서류 |
| **미수금** | 청구했으나 아직 입금되지 않은 금액 |

### 기사 정산

| 용어 | 설명 |
|------|------|
| **기사 운임** | 기사에게 지급하는 운송료 |
| **산재보험** | 기사 산재보험료 (운임의 약 0.44%) |
| **빠른 지급** | 정상 지급일보다 빨리 지급 (수수료 1.5%) |
| **지급 예정일** | 기사에게 운임이 입금될 예정 날짜 |

## 시스템 상태

### 배차 상태

| 상태 | 영문 | 설명 |
|------|------|------|
| 대기중 | pending | 주문이 생성되었으나 아직 처리 전 |
| 배차요청 | requested | 기사 배정이 진행 중 |
| 정산중 | accounting | 운송 완료, 정산 처리 중 |
| 출금대기 | withdraw | 기사 지급 대기 중 |
| 완료 | finished | 모든 처리 완료 |
| 취소 | canceled | 배차 취소됨 |

### 청구 상태

| 상태 | 설명 |
|------|------|
| 미청구 | 아직 청구서에 포함되지 않음 |
| 청구전 | 청구월은 지정됐으나 세금계산서 미발행 |
| 입금전 | 세금계산서 발행됨, 입금 대기 |
| 완료 | 입금 확인됨 |
| 미수 | 입금 기한 초과 |

### 세금계산서 상태

| 상태 | 영문 | 설명 |
|------|------|------|
| 대기 | pending | 처리 대기 중 |
| 자동매칭 | automatch | 배차와 자동 연결됨 |
| 검토필요 | review | 수동 확인 필요 |
| 승인 | approved | 처리 완료 |
| 거절 | declined | 거절됨 |
| 취소 | canceled | 취소됨 |

| 권한 | 영문 | 설명 |
|------|------|------|

|------|------|
| **정담당자** | 업체의 주 담당자 |
| **부담당자** | 업체의 보조 담당자 |
