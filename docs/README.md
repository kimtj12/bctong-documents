## 시스템 요약

### 배차통이란?
**화주**(화물을 보내는 업체)와 **차주**(화물을 운송하는 기사)를 연결하여 화물 운송을 중개하는 플랫폼입니다.

### 핵심 기능
- 배차 요청 및 관리
- 기사 배정 및 운송 추적
- 청구서 발행 및 정산
- 세금계산서 자동 처리
- 실시간 알림 (Slack, 카카오톡)

### 사용자 유형
| 유형 | 설명 |
|------|------|
| 화주 (user) | 화물 운송을 의뢰하는 업체 |
| 관리자 (admin) | 시스템 전체를 관리하는 운영자 |
| 파트너 (partner) | 배차를 담당하는 협력사 |
| 프리랜서 (freelancer) | 영업 프리랜서 |
