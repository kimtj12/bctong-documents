# 관리자 기능

관리자(admin, partner, freelancer)가 사용할 수 있는 기능을 설명합니다.

---

## 기능 개요

| 카테고리 | 기능 | Admin | Partner | Freelancer |
|----------|------|:-----:|:-------:|:----------:|
| **배차 관리** | 배차상황실 | ✓ | ✓ | ✓ |
| | 선착불 배차 | ✓ | ✓ | - |
| **화주 정산** | 업체별 거래내역 | ✓ | △ | △ |
| | 청구내역 생성 | ✓ | - | - |
| | 수금/미수금 관리 | ✓ | - | - |
| | 매출 세금계산서 | ✓ | - | - |
| **차주 관리** | 매입 세금계산서 | ✓ | - | - |
| | 지급 관리 | ✓ | - | - |
| | 기사 블랙리스트 | ✓ | ✓ | - |
| **관리자 메뉴** | 계약처 관리 | ✓ | - | - |
| | 사용자 관리 | ✓ | - | - |
| | 마일리지 | ✓ | - | - |
| | 상품권 설정 | ✓ | - | - |
| **분석** | 배차 추이 | ✓ | - | - |
| | 거리별 운임 | ✓ | ✓ | - |

*△ = 제한적 접근*

## 1. 배차 관리

### 배차상황실

실시간으로 배차를 관리하는 핵심 화면입니다.

#### 주요 기능

| 기능 | 설명 |
|------|------|
| 배차 조회 | 전체 배차 목록 조회 |
| 상태 필터 | 대기중/배차요청/정산중 등 |
| 기사 배정 | 기사 정보 입력 및 배정 |
| 배차 수정 | 배차 정보 변경 |
| 배차 취소 | 취소 처리 |
| 24시콜 연동 | 외부 플랫폼 등록 |

#### 배차 목록 정보

| 항목 | 설명 |
|------|------|
| 배차번호 | 고유 식별자 |
| 상태 | 현재 처리 단계 |
| 거래처 | 의뢰 업체 |
| 상/하차지 | 출발/도착 장소 |
| 상차일 | 상차 예정일 |
| 차량 | 차종/톤수 |
| 운송료 | 화주 청구 금액 |
| 기사 운임 | 기사 지급 금액 |
| 기사 정보 | 기사명, 차량번호 |
| 담당자 | 배차 담당 관리자 |

#### 기사 배정

기사 정보를 입력하여 배차를 완료합니다.

| 입력 항목 | 필수 | 설명 |
|----------|:----:|------|
| 기사명 | ✓ | 운송 기사 이름 |
| 연락처 | ✓ | 기사 전화번호 |
| 차량번호 | ✓ | 배정 차량 번호 |
| 기사 운임 | ✓ | 기사에게 지급할 금액 |
| 은행/계좌 | - | 정산 계좌 정보 |

---

### 선착불 배차

일반 배차와 다른 정산 방식의 배차를 관리합니다.

| 상태 | 설명 |
|------|------|
| 수수료요청 | 선착불 배차 요청됨 |
| 수수료완료 | 처리 완료 |
| 수수료취소 | 취소됨 |

---

## 2. 화주 정산 관리

### 업체별 거래내역

거래처별 배차 및 정산 현황을 조회합니다.

#### 조회 정보

| 항목 | 설명 |
|------|------|
| 거래처명 | 업체 이름 |
| 배차 건수 | 기간 내 배차 수 |
| 총 운송료 | 화주 청구 총액 |
| 총 기사 운임 | 기사 지급 총액 |
| 수익 | 운송료 - 기사 운임 |

---

### 청구내역 생성

월별 청구서를 생성합니다.

#### 프로세스

1. **미청구 주문 조회**
   - 기간 선택
   - 거래처별 미청구 건 확인

2. **청구월 지정**
   - 대상 배차 선택
   - 청구월 입력 (예: 2024-01)
   - 일괄 적용

3. **청구서 생성**
   - 거래처별 청구서 생성
   - 금액 합산

4. **세금계산서 발행**
   - Popbill 연동 발행
   - 이메일 자동 발송

---

### 청구 및 수금 내역

발행된 청구서와 입금 현황을 관리합니다.

#### 조회 정보

| 항목 | 설명 |
|------|------|
| 청구월 | 청구 기준 월 |
| 거래처 | 청구 대상 업체 |
| 청구액 | 총 청구 금액 |
| 입금액 | 입금된 금액 |
| 잔액 | 미입금 금액 |
| 상태 | 입금전/부분입금/완료/미수 |

#### 입금 처리

| 입력 항목 | 설명 |
|----------|------|
| 입금일 | 입금 날짜 |
| 입금액 | 입금 금액 |
| 메모 | 참고 사항 |

---

### 미수금 관리

입금 기한이 지난 청구건을 관리합니다.

#### 미수금 목록

| 항목 | 설명 |
|------|------|
| 거래처 | 미수 업체 |
| 청구월 | 미수 발생 월 |
| 미수액 | 미입금 금액 |
| 경과일 | 기한 초과 일수 |
| 담당자 | 거래처 담당 관리자 |

---

### 매출 전자세금계산서

발행한 세금계산서를 관리합니다.

#### 조회/관리

| 기능 | 설명 |
|------|------|
| 목록 조회 | 발행 내역 조회 |
| 상세 보기 | 세금계산서 상세 |
| 발행 취소 | 취소 처리 |

---

## 3. 차주 관리

### 매입 전자세금계산서

기사로부터 받은 세금계산서를 관리합니다.

#### 세금계산서 상태

| 상태 | 설명 |
|------|------|
| 대기 | 처리 대기 |
| 자동매칭 | 배차와 자동 연결됨 |
| 검토필요 | 수동 확인 필요 |
| 승인 | 처리 완료 |
| 거절 | 거절됨 |

#### 매칭 기능

- 차량번호, 금액, 날짜로 자동 매칭
- 수동 매칭 지원
- 매칭 해제 가능

---

### 지급 관리

기사 운임 지급을 관리합니다.

#### 지급 예정 목록

| 항목 | 설명 |
|------|------|
| 기사명 | 지급 대상 |
| 차량번호 | 기사 차량 |
| 배차 건수 | 지급 대상 건수 |
| 지급 예정액 | 총 지급 금액 |
| 지급 예정일 | 지급 날짜 |

#### 일괄 처리

- 지급 목록 엑셀 다운로드
- 일괄 완료 처리

---

### 기사 블랙리스트

문제 기사를 관리합니다.

#### 등록 정보

| 항목 | 설명 |
|------|------|
| 기사명 | 기사 이름 |
| 연락처 | 전화번호 |
| 차량번호 | 차량 번호 |
| 유형 | 블랙리스트/화이트리스트 |
| 사유 | 등록 이유 |

---

## 4. 관리자 메뉴

### 계약처(거래처) 관리

거래처 정보를 등록하고 관리합니다.

#### 거래처 정보

| 구분 | 항목 |
|------|------|
| 기본 정보 | 회사명, 사업자번호, 대표자, 주소 |
| 연락처 | 전화, 팩스, 이메일 |
| 세금계산서 정보 | 상호, 업태, 종목, 담당자 |
| 담당자 배정 | 정담당자, 부담당자 |
| 계약 서류 | 사업자등록증, 보험증권, 계약서 |
| 가격표 | 구간별 단가표 URL |

---

### 상/하차지 관리

모든 거래처의 상/하차지를 관리합니다.

- 거래처별 상/하차지 조회
- 일괄 등록/수정
- 위치 정보 관리

---

### 사용자 계정 관리

시스템 사용자를 관리합니다.

#### 사용자 정보

| 항목 | 설명 |
|------|------|
| 아이디 | 로그인 이메일 |
| 이름 | 사용자 실명 |
| 연락처 | 전화번호 |
| 권한 | admin/partner/freelancer/user |
| 소속 거래처 | 해당 업체 |
| 활성 상태 | 활성/비활성 |

#### 권한 변경

- 사용자 권한 승격/강등
- 계정 활성화/비활성화

---

### 마일리지 내역

모든 사용자의 포인트 내역을 관리합니다.

- 사용자별 포인트 조회
- 적립/사용 내역 확인
- 출금 요청 처리

---

### 상품권 설정

마일리지 관련 설정을 관리합니다.

---

### 알림센터

시스템 알림 내역을 관리합니다.

- 알림 목록 조회
- 처리 상태 확인
- 알림 상세 보기

---

## 5. 분석 기능

### 배차 추이 분석

담당자별, 기간별 배차 현황을 분석합니다.

#### 분석 지표

| 지표 | 설명 |
|------|------|
| 배차 건수 | 기간 내 처리 건수 |
| 배차 매출 | 총 운송료 |
| 수익률 | (운송료 - 기사운임) / 운송료 |
| 업체 수 | 거래 업체 수 |

#### 시각화

- 일별 추이 그래프
- 담당자별 비교
- 지분율 파이 차트

---

### 거리별 운임 단가

거리 구간별 평균 운임을 분석합니다.

#### 분석 결과

| 거리 구간 | 1톤 평균 | 5톤 평균 | ... |
|----------|----------|----------|-----|
| 0-10km | 50,000원 | 80,000원 | ... |
| 10-20km | 60,000원 | 100,000원 | ... |
| ... | ... | ... | ... |

---

### 가격 분석

기존 배차 데이터를 분석하여 가격 책정에 활용합니다.
