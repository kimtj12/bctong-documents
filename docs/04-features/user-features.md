# 화주(사용자) 기능

화주 업체 직원(일반 사용자)이 사용할 수 있는 기능을 설명합니다.

---

## 기능 개요

화주는 다음 기능을 사용할 수 있습니다:

| 카테고리 | 기능 |
|----------|------|
| **배차** | 신규 배차 요청, 배차 내역 조회 |
| **정산** | 거래명세서 확인 |
| **관리** | 거래처(상/하차지) 관리, 상차품 관리 |
| **마일리지** | 포인트 조회, 출금 |
| **계정** | 내 회사 정보 관리 |

---

## 1. 배차 요청

### 신규 배차 요청

새로운 화물 운송을 요청합니다.

#### 입력 정보

**출발 정보 (상차지)**

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 상차지명 | ✓ | 출발 장소 이름 |
| 주소 | ✓ | 상세 주소 |
| 연락처 | ✓ | 현장 담당자 전화번호 |
| 상차 일시 | ✓ | 상차 예정 일시 |
| 상차 유형 | ✓ | 당상/내상/월상/기타 |

**도착 정보 (하차지)**

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 하차지명 | ✓ | 도착 장소 이름 |
| 주소 | ✓ | 상세 주소 |
| 연락처 | ✓ | 현장 담당자 전화번호 |
| 하차 일시 | ✓ | 하차 예정 일시 |
| 하차 유형 | ✓ | 당착/내착/기타 |

**화물 정보**

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 상차품명 | ✓ | 화물 종류/내용 |
| 무게 | - | 화물 중량 |
| 이미지 | - | 화물 사진 |

**차량 정보**

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 차량 종류 | ✓ | 카고/윙바디/탑차 등 |
| 톤수 | ✓ | 0.5톤 ~ 25톤 |
| 상차 방법 | - | 지게차/수작업 등 |
| 하차 방법 | - | 지게차/수작업 등 |

**옵션**

| 항목 | 설명 |
|------|------|
| 왕복 | 왕복 운송 여부 |
| 혼적 | 다른 화물과 혼재 가능 여부 |
| 인수증 필요 | 인수증 회수 필요 여부 |

#### 운임 자동 계산

- 출발/도착 주소 입력 시 거리 자동 계산
- 등록된 단가표가 있으면 운임 자동 적용
- 없으면 기본 운임 표시

#### 빠른 배차

자주 사용하는 상/하차지를 선택하여 빠르게 배차할 수 있습니다.

---

### 배차 내역 조회

자사의 배차 내역을 조회합니다.

#### 조회 조건

| 조건 | 설명 |
|------|------|
| 기간 | 시작일 ~ 종료일 |
| 상태 | 전체/배차중/완료/취소 |
| 검색어 | 상/하차지명, 기사명 등 |

#### 조회 결과

| 정보 | 설명 |
|------|------|
| 배차번호 | 고유 번호 |
| 상태 | 현재 처리 단계 |
| 상차지/하차지 | 출발/도착 장소 |
| 상차일시 | 상차 예정일 |
| 차량 | 차량 종류/톤수 |
| 운송료 | 청구 금액 |
| 기사 정보 | 기사명, 차량번호 (배정 후) |

#### 상세 보기

배차 건을 클릭하면 상세 정보를 확인할 수 있습니다:
- 전체 배차 정보
- 기사 정보 (배정된 경우)
- 운송료 상세
- 관련 서류 (인수증, 통관서류 등)

#### 엑셀 다운로드

조회 결과를 엑셀 파일로 다운로드할 수 있습니다.

---

## 2. 거래명세서

월별 거래 내역을 확인합니다.

### 조회 조건

| 조건 | 설명 |
|------|------|
| 연도/월 | 조회할 청구월 선택 |
| 상태 | 청구전/입금전/완료/미수 |

### 거래명세서 내용

| 정보 | 설명 |
|------|------|
| 청구월 | 청구 기준 월 |
| 배차 목록 | 해당 월에 포함된 배차 건 |
| 공급가 | 세금 제외 금액 |
| 부가세 | 세금 금액 |
| 합계 | 총 청구 금액 |
| 상태 | 청구/입금 상태 |

### 상태별 의미

| 상태 | 설명 |
|------|------|
| 청구전 | 아직 세금계산서 미발행 |
| 입금전 | 세금계산서 발행됨, 입금 대기 |
| 완료 | 입금 확인됨 |
| 미수 | 입금 기한 초과 |

---

## 3. 거래처 관리

자주 사용하는 상/하차지를 등록하고 관리합니다.

### 상/하차지 등록

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 장소명 | ✓ | 상/하차지 이름 |
| 유형 | ✓ | 상차지/하차지 구분 |
| 주소 | ✓ | 상세 주소 |
| 상세주소 | - | 추가 위치 정보 |
| 담당자명 | - | 현장 담당자 |
| 연락처 | - | 담당자 전화번호 |
| 메모 | - | 참고 사항 |

### 상/하차지 활용

- 배차 요청 시 등록된 장소 선택
- 주소, 연락처 자동 입력
- 배차 시간 단축

### 목록 조회

| 정보 | 설명 |
|------|------|
| 장소명 | 상/하차지 이름 |
| 유형 | 상차지/하차지 |
| 주소 | 등록된 주소 |
| 담당자 | 담당자 정보 |

---

## 4. 상차품 관리

자주 운송하는 화물 정보를 등록합니다.

### 상차품 등록

| 항목 | 설명 |
|------|------|
| 상차품명 | 화물 이름/종류 |
| 이미지 | 화물 사진 |

### 상차품 활용

- 배차 요청 시 등록된 상차품 선택
- 이미지 자동 첨부
- 반복 입력 감소

---

## 5. 마일리지

포인트 적립 및 사용 내역을 관리합니다.

### 포인트 조회

| 정보 | 설명 |
|------|------|
| 현재 포인트 | 사용 가능한 포인트 잔액 |
| 적립 내역 | 포인트 적립 기록 |
| 사용 내역 | 포인트 사용 기록 |

### 포인트 출금

일정 금액 이상(30,000P)이면 출금 신청할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 출금 금액 | 출금할 포인트 |
| 계좌 정보 | 입금받을 계좌 |

---

## 6. 내 회사 관리

소속 회사 정보를 확인하고 수정합니다.

### 회사 정보

| 정보 | 수정 가능 |
|------|:--------:|
| 회사명 | - |
| 사업자번호 | - |
| 대표자명 | - |
| 주소 | ✓ |
| 연락처 | ✓ |
| 이메일 | ✓ |

### 담당자 정보

세금계산서 수신 담당자 정보를 관리합니다.

---

## 7. 자료실

바른물류 공지사항 및 서식 자료를 확인합니다.

### 내용

| 구분 | 설명 |
|------|------|
| 공지사항 | 서비스 관련 안내 |
| 서식 자료 | 업무 관련 문서 양식 |

---

## 화면 구성

### 메인 화면 (대시보드)

- 신규 배차 요청 바로가기
- 최근 배차 내역
- 이번 달 거래 현황

### 네비게이션 메뉴

| 메뉴 | 기능 |
|------|------|
| 배차요청 | 신규 배차 등록 |
| 배차내역 | 배차 조회 |
| 거래명세서 | 청구 내역 |
| 거래처관리 | 상/하차지 관리 |
| 상차품관리 | 화물 정보 관리 |
| 마일리지 | 포인트 관리 |
| 내회사 | 회사 정보 |
| 자료실 | 공지/서식 |
