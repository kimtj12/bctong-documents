# 사용자 화면 가이드

화주(일반 사용자)가 사용하는 화면을 설명합니다.

---

## 화면 구성 개요

```
┌──────────────────────────────────────────────┐
│  헤더 (로고, 알림, 계정 메뉴)                    │
├──────────┬───────────────────────────────────┤
│          │                                   │
│  사이드   │        메인 콘텐츠 영역             │
│  메뉴     │                                   │
│          │                                   │
│          │                                   │
└──────────┴───────────────────────────────────┘
```

---

## 1. 로그인/회원가입

### 로그인 화면

| 요소 | 설명 |
|------|------|
| 이메일 입력 | 로그인 아이디 |
| 비밀번호 입력 | 비밀번호 |
| 로그인 버튼 | 로그인 실행 |
| 비밀번호 찾기 | 비밀번호 재설정 |
| 회원가입 | 신규 가입 |

### 회원가입 화면

| 입력 항목 | 필수 | 설명 |
|----------|:----:|------|
| 이메일 | ✓ | 로그인 아이디로 사용 |
| 비밀번호 | ✓ | 6자 이상 |
| 비밀번호 확인 | ✓ | 비밀번호 재입력 |
| 이름 | ✓ | 실명 |
| 연락처 | ✓ | 휴대폰 번호 |
| 회사명 | - | 소속 회사 |

---

## 2. 대시보드 (메인)

로그인 후 처음 보이는 화면입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  신규 배차 요청 버튼                          │
├─────────────────────────────────────────────┤
│  최근 배차 내역                              │
│  ┌─────┬─────┬─────┬─────┬─────┐           │
│  │번호 │상태 │상차지│하차지│일시  │           │
│  ├─────┼─────┼─────┼─────┼─────┤           │
│  │1001 │완료 │인천  │서울  │01/15│           │
│  │1002 │배차중│부산  │대전  │01/16│           │
│  └─────┴─────┴─────┴─────┴─────┘           │
├─────────────────────────────────────────────┤
│  이번 달 현황                                │
│  배차: 25건 | 운송료: 5,000,000원             │
└─────────────────────────────────────────────┘
```

### 주요 기능

| 영역 | 기능 |
|------|------|
| 신규 배차 버튼 | 배차 요청 화면으로 이동 |
| 최근 배차 | 최근 5건 배차 목록 |
| 이번 달 현황 | 월간 배차 통계 |

## 3. 배차 요청 화면

새로운 배차를 요청하는 화면입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  배차 요청                                   │
├─────────────────────────────────────────────┤
│  출발 정보 (상차지)                           │
│  ┌─────────────────────────────────────┐    │
│  │ 상차지명: [          ] [등록된 장소 ▼]│    │
│  │ 주소:    [                    ] 🔍  │    │
│  │ 연락처:  [          ]               │    │
│  │ 상차일시: [날짜] [시간] [당상 ▼]     │    │
│  └─────────────────────────────────────┘    │
├─────────────────────────────────────────────┤
│  도착 정보 (하차지)                           │
│  ┌─────────────────────────────────────┐    │
│  │ (출발 정보와 동일한 형식)              │    │
│  └─────────────────────────────────────┘    │
├─────────────────────────────────────────────┤
│  화물/차량 정보                              │
│  ┌─────────────────────────────────────┐    │
│  │ 상차품명: [          ] [등록된 품목 ▼]│    │
│  │ 차량종류: [카고 ▼] 톤수: [5톤 ▼]      │    │
│  │ □ 왕복  □ 혼적  □ 인수증 필요        │    │
│  └─────────────────────────────────────┘    │
├─────────────────────────────────────────────┤
│  운송료 정보                                 │
│  거리: 85.3km | 예상시간: 1시간 30분          │
│  운송료: 500,000원                           │
├─────────────────────────────────────────────┤
│  [취소] [배차 요청]                          │
└─────────────────────────────────────────────┘
```

### 입력 흐름

1. **출발 정보 입력**
   - 등록된 상차지 선택 또는 직접 입력
   - 주소 검색 (🔍 버튼)
   - 상차 일시 및 유형 선택

2. **도착 정보 입력**
   - 등록된 하차지 선택 또는 직접 입력

3. **화물/차량 정보**
   - 상차품명 입력 또는 선택
   - 차량 종류 및 톤수 선택
   - 옵션 체크

4. **운송료 확인**
   - 자동 계산된 거리/운송료 확인

5. **배차 요청**
   - 버튼 클릭으로 요청 완료

## 4. 배차 내역 화면

과거 배차 내역을 조회합니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  배차 내역                                   │
├─────────────────────────────────────────────┤
│  검색 조건                                   │
│  기간: [시작일] ~ [종료일]                    │
│  상태: [전체 ▼]  검색어: [        ] [검색]   │
├─────────────────────────────────────────────┤
│  ┌─────┬─────┬─────────┬─────────┬─────┐   │
│  │번호 │상태 │상차지    │하차지    │상차일│   │
│  ├─────┼─────┼─────────┼─────────┼─────┤   │
│  │1001 │완료 │인천항    │서울 강남 │01/15│   │
│  │1002 │배차중│부산항   │대전 유성 │01/16│   │
│  │1003 │대기중│광양항   │서울 마포 │01/17│   │
│  └─────┴─────┴─────────┴─────────┴─────┘   │
├─────────────────────────────────────────────┤
│  [엑셀 다운로드]           페이지: ◀ 1 2 3 ▶ │
└─────────────────────────────────────────────┘
```

### 상태별 색상

| 상태 | 색상 | 의미 |
|------|------|------|
| 대기중 | 회색 | 처리 전 |
| 배차요청 | 파랑 | 기사 배정 중 |
| 정산중 | 주황 | 운송 완료, 정산 중 |
| 완료 | 초록 | 모든 처리 완료 |
| 취소 | 빨강 | 취소됨 |

## 5. 거래명세서 화면

월별 청구 내역을 확인합니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  거래명세서                                  │
├─────────────────────────────────────────────┤
│  연도: [2024 ▼]  상태: [전체 ▼]              │
├─────────────────────────────────────────────┤
│  ┌─────┬─────────┬───────────┬──────┐      │
│  │청구월│공급가    │합계       │상태  │      │
│  ├─────┼─────────┼───────────┼──────┤      │
│  │01월 │5,000,000│5,500,000  │완료  │      │
│  │02월 │3,200,000│3,520,000  │입금전│      │
│  │03월 │4,100,000│4,510,000  │청구전│      │
│  └─────┴─────────┴───────────┴──────┘      │
└─────────────────────────────────────────────┘
```

### 상세 보기

월을 클릭하면 해당 월의 상세 내역을 확인할 수 있습니다:

- 포함된 배차 목록
- 각 배차별 금액
- 총액 합계
