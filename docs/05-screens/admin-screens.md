# 관리자 화면 가이드

관리자가 사용하는 화면을 설명합니다.

---

## 화면 구성 개요

관리자 화면은 일반 사용자 화면보다 더 많은 메뉴와 기능을 제공합니다.

```
┌──────────────────────────────────────────────┐
│  헤더 (로고, 알림, 계정)                       │
├──────────┬───────────────────────────────────┤
│          │                                   │
│  관리자   │        메인 콘텐츠 영역             │
│  메뉴     │                                   │
│  (확장)   │                                   │
│          │                                   │
└──────────┴───────────────────────────────────┘
```

---

## 1. 배차상황실

실시간 배차를 관리하는 핵심 화면입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
│  배차상황실                      [+ 신규 배차]│
├─────────────────────────────────────────────┤
│  필터: [전체상태 ▼] [전체거래처 ▼] [담당자 ▼]  │
│  기간: [시작일] ~ [종료일]  검색: [    ] [🔍] │
├─────────────────────────────────────────────┤
│  상태별 탭: [전체] [대기] [배차중] [정산중] ...│
├─────────────────────────────────────────────┤
│  ┌────┬────┬─────┬─────┬────┬────┬────┐    │
│  │번호│상태│거래처│상/하차│운송료│기사비│기사│    │
│  ├────┼────┼─────┼─────┼────┼────┼────┤    │
│  │1001│완료│한국물류│인천→서울│50만│45만│홍길동│    │
│  │1002│배차│대한운송│부산→대전│35만│ - │미정│    │
│  └────┴────┴─────┴─────┴────┴────┴────┘    │
├─────────────────────────────────────────────┤
│  [일괄처리 ▼] [엑셀] [24시콜]    페이지: ◀ 1 ▶│
└─────────────────────────────────────────────┘
```

### 배차 상세/수정 모달

배차를 클릭하면 상세 정보와 수정 화면이 표시됩니다:

```
┌─────────────────────────────────────────────┐
│  배차 #1001                            [X]  │
├─────────────────────────────────────────────┤
│  [기본정보] [기사정보] [정산정보] [이력]       │
├─────────────────────────────────────────────┤
│  기본정보 탭                                 │
│  ┌─────────────────────────────────────┐    │
│  │ 거래처: (주)한국물류                  │    │
│  │ 상차지: 인천항 3부두                  │    │
│  │ 하차지: 서울 강남 물류센터             │    │
│  │ 상차일: 01/15 09:00 (당상)           │    │
│  │ 차량: 5톤 윙바디                     │    │
│  │ 운송료: 500,000원                   │    │
│  └─────────────────────────────────────┘    │
├─────────────────────────────────────────────┤
│  [삭제] [취소]                    [저장]     │
└─────────────────────────────────────────────┘
```

### 기사 배정

기사정보 탭에서 기사를 배정합니다:

```
┌─────────────────────────────────────────────┐
│  기사정보 탭                                 │
│  ┌─────────────────────────────────────┐    │
│  │ 기사명: [홍길동        ]              │    │
│  │ 연락처: [010-1111-2222 ]              │    │
│  │ 차량번호: [12가 3456   ]              │    │
│  │ 기사 운임: [450,000    ]원            │    │
│  │ 추가비용: [     0     ]원             │    │
│  │ 사유: [               ]               │    │
│  ├─────────────────────────────────────┤    │
│  │ 정산 정보                            │    │
│  │ 은행: [국민은행 ▼]                    │    │
│  │ 계좌: [123-456-78901234]             │    │
│  │ 지급일: [2024-01-20]                 │    │
│  │ □ 빠른 지급 (수수료 1.5%)             │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```
